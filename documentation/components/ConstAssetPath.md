# Constant Asset Path

Provides a constant file or asset path that you can set\.<br/><br/>Use this to provide fixed paths to textures, models, configuration files, or other assets\.

## Component Information

- **Name:** `ConstAssetPath`
- **UI Name:** Constant Asset Path
- **Version:** 1
- **Categories:** Constants

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| value | Value | AssetPath | Input | "" | No | 

### Value

The constant file or asset path\.


## Usage Notes

This component is part of the RTX Remix graph system. It is intended for use in the Remix Toolkit and Runtime only.
