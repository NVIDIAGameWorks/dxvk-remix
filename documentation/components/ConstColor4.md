# Constant Color4

Provides a constant RGBA color \(Red, Green, Blue, Alpha\) that you can set\.<br/><br/>Use this for fixed colors with transparency\. Each channel ranges from 0\.0 to 1\.0\.

## Component Information

- **Name:** `ConstColor4`
- **UI Name:** Constant Color4
- **Version:** 1
- **Categories:** Constants

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| value | Value | Float4 | Input | \[0\.0, 0\.0, 0\.0, 0\.0\] | No | 

### Value

The constant RGBA color \(Red, Green, Blue, Alpha\)\.
