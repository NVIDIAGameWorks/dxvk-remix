# Constant Prim

Provides a constant reference to a scene object \(prim\) that you can set\.<br/><br/>Use this to provide fixed references to meshes, lights, cameras, or other scene elements\.

## Component Information

- **Name:** `ConstPrim`
- **UI Name:** Constant Prim
- **Version:** 1
- **Categories:** Constants

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| value | Value | Prim | Input | None | No | 

### Value

The constant prim reference\.


## Usage Notes

This component is part of the RTX Remix graph system. It is intended for use in the Remix Toolkit and Runtime only.
