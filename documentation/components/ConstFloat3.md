# Constant Float3

Provides a constant 3D vector \(three numbers: X, Y, Z\) that you can set\.<br/><br/>Use this for fixed 3D positions, RGB colors, directions, or any set of three values\.

## Component Information

- **Name:** `ConstFloat3`
- **UI Name:** Constant Float3
- **Version:** 1
- **Categories:** Constants

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| value | Value | Float3 | Input | \[0\.0, 0\.0, 0\.0\] | No | 

### Value

The constant 3D vector \(X, Y, Z\)\.
