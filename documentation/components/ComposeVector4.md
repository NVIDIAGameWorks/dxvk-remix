# Compose Vector4

Combines four separate numbers into a single Vector4\.

## Component Information

- **Name:** `ComposeVector4`
- **UI Name:** Compose Vector4
- **Version:** 1
- **Categories:** Transform

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| x | X | Float | Input | 0\.0 | No | 
| y | Y | Float | Input | 0\.0 | No | 
| z | Z | Float | Input | 0\.0 | No | 
| w | W | Float | Input | 0\.0 | No | 

### X

The X component of the vector \(Red channel\)\.


### Y

The Y component of the vector \(Green channel\)\.


### Z

The Z component of the vector \(Blue channel\)\.


### W

The W component of the vector \(Alpha channel\)\.


## Output Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| vector | Vector | Float4 | Output | \[0\.0, 0\.0, 0\.0, 0\.0\] | No | 

### Vector

The composed vector\.

This component is part of the RTX Remix graph system. It is intended for use in the Remix Toolkit and Runtime only.
