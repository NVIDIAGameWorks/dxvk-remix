# Compose Vector2

Combines two separate numbers into a single Vector2\.

## Component Information

- **Name:** `ComposeVector2`
- **UI Name:** Compose Vector2
- **Version:** 1
- **Categories:** Transform

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| x | X | Float | Input | 0\.0 | No | 
| y | Y | Float | Input | 0\.0 | No | 

### X

The X component of the vector\.


### Y

The Y component of the vector\.


## Output Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| vector | Vector | Float2 | Output | \[0\.0, 0\.0\] | No | 

### Vector

The composed vector\.
