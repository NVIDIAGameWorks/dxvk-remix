# Round

Rounds a value to the nearest integer\.<br/><br/>Rounds to the nearest whole number\. For example: 1\.4 becomes 1\.0, 1\.5 becomes 2\.0, 1\.6 becomes 2\.0\.

## Component Information

- **Name:** `Round`
- **UI Name:** Round
- **Version:** 1
- **Categories:** Transform

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| input | Input | Float | Input | 0\.0 | No | 

### Input

The value to round\.


## Output Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| result | Result | Float | Output | 0\.0 | No | 

### Result

The input value rounded to the nearest integer\.
