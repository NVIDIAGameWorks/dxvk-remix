# Ceil

Rounds a value up to the next integer\.<br/><br/>Returns the smallest integer greater than or equal to the input\. For example: 1\.1 becomes 2\.0, 1\.9 becomes 2\.0, \-1\.1 becomes \-1\.0\.

## Component Information

- **Name:** `Ceil`
- **UI Name:** Ceil
- **Version:** 1
- **Categories:** Transform

## Input Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| input | Input | Float | Input | 0\.0 | No | 

### Input

The value to round up\.


## Output Properties

| Property | Display Name | Type | IO Type | Default Value | Optional |
|----------|--------------|------|---------|---------------|----------|
| result | Result | Float | Output | 0\.0 | No | 

### Result

The input value rounded up to the next integer\.
